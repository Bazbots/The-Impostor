What is it?

The Impostor is a bot made to play Among Us on Discord!

Current commands:

Use the prefix "$"

You can also use $help or $commands for help

invite - sends an invite link for you to add the bot

about - tells you about the bot

version - reports back the current version of the bot

prefix - tells you the current prefix, you will be able to change it soon

website - sends a link to this page

vote - when the bot is ready, we will register it to top.gg!

servers - returns the amount of servers the bot is in

creator - gives a bit of info about the creator

github - sends a link to our github repo

feedback - get a link to a feedback form

ping - get back your current ping

eject {user} {role} - eject the person you provide!

sus - that is very sus

kill {username} - die potatoes!

host - start a game with you as host

mute - host command only

unmute - host command only

dead - mute yourself for the rest of the game

sponsors - view all of our current sponsors!

scan - prove you are not the impostor

download - download data and complete your tasks!

weatherdownload - go to Mira HQ and get the current weather!

inspect - inspect the medbay sample!
